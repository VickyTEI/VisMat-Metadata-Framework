Visual Materials Metadata Framework

The framework outlines metadata relating to specific information categories relevant to cataloguing an object from the Visual collection. It is recognised it may not be possible to provide content for each category, but an ideal record should contain as many as available information allows, with those included in the Inventory minimal viable record + Physical description, Access status & conditions, reproduction conditions and condition as mandatory.
